Book Management Web Application
Overview

This project is a simple Book Management Web Application that allows users to add, view, update, and delete books.
It is built using:

Frontend: Angular

Backend: ASP.NET Core Web API (C#)

Data Storage: In-memory list (no database)

The application demonstrates full-stack CRUD operations using a RESTful API.

Features

Display a list of books

Add new books with Title, Author, ISBN, and Publication Date

Update existing books

Delete books

Responsive and user-friendly interface

Technologies Used

Angular

TypeScript

HTML, CSS, JavaScript

ASP.NET Core Web API

C#
